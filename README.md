## Mô tả bài tập

Thực hiện chuyển đổi từ code C sang Assembly cho 2 bài toán:

1. Chuyển chuỗi `"Ky thuat VXL"` thành chữ thường và in ra màn hình
2. Chuyển chuỗi `"125"` thành số nguyên và lưu vào biến
